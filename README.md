# Agora

Agora is a centralized event and community platform designed specifically for Boston University students. It aims to solve the fragmentation of campus life by aggregating events, facilitating community discovery, and providing robust tools for student organizations.

## Features

- **Centralized Event Feed**: Automatically scrapes and aggregates events from TerrierCentral and other sources.
- **Advanced Filtering**: Find events based on interests, tags, organizations, and "vibes".
- **Community Management**: Create and join interest-based communities.
- **Student Org Tools**: Manage event pages, RSVPs, ticketing, and track engagement.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Database**: [PostgreSQL](https://www.postgresql.org/) (with [Drizzle ORM](https://orm.drizzle.team/))

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
